## Setup
```sh
$ git clone https://github.com/nightguard322/php-project-lvl1.git
$ cd php-project-lvl1
$ make install
```

## Run games:

Calc Game: 
```sh
make brain-calc
```
Even Game: 
```sh
make brain-even
```
GCD Game: 
```sh
make brain-gcd
```
Prime Game: 
```sh
make brain-prime
```
Progression Game:
```sh 
make brain-progression
```
